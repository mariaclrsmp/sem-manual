# Sem Manual — Assistente Doméstico

## Conceito

App React Native/Expo para quem está aprendendo a morar sozinho.
Foco em gamificação, assistente proativo e guias domésticos práticos.
NÃO é um app de listas — é um companheiro de evolução na vida adulta.

## Stack

- Expo SDK 51 + Expo Router v3 + TypeScript
- NativeWind (Tailwind para RN)
- Zustand + React Query
- Supabase (Auth + DB + Edge Functions)
- Fonte: Nunito
